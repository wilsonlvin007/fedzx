# Roadmap

## Phase 1: Manual Backoffice (MVP)

- Admin auth
- Articles CRUD + publish workflow
- Page modules CRUD (home sections)
- Recommendations CRUD
- Public JSON export endpoints
- Minimal audit log

## Phase 2: Semi-Automation

- RSS import jobs (store raw feed entries)
- Import mapping UI (select, merge, discard)
- AI-assisted draft generation (summaries, titles, metadata)
- Human approval gates + diff/preview

## Phase 3: AI Operations

- Scheduled ingestion + monitoring
- Automated draft generation + quality checks
- Auto-publish with guardrails + rollback
- Observability: job traces, alerts, content anomaly detection
